# image-gen2 図解プロンプト案

## 目的

今日のAIMTGの要点を、FRACTAL WORKOUTの社内共有用に1枚の図解画像へ整理する。

## プロンプト

FRACTAL WORKOUTのAIMTG議事録を整理した、横長16:9の日本語ビジネス図解。白背景、黒、チャコール、シルバーグレーを基調にした高級感のあるミニマルデザイン。上部に大きく「AIMTG 活用マップ」とタイトル。中央に「会話を記録 → AIで構造化 → 施策へ展開」という太い横フロー。下部に5つのカテゴリカードを横並びで配置。

1. 医療機関連携：クリニックの運動説明負荷を減らす、提携希望医療機関向け見学・相談
2. 展示会施策：診断フォーム、会社名入力、QR照合、相談予約、特典
3. BODY PALETTE：アンケート、管理画面、AIコメント、レポート化
4. ナレッジ化：Zoom/Meet録音、文字起こし、商談フォルダ、PR/LP/営業資料へ再利用
5. LP制作：メディカルフィットネス、ピラティス、男女別FV、写真差し替え

右下に小さく「次回まで：診断フォーム改修 / 医療連携記事案 / 展示会特典整理」と入れる。人物写真は使わず、抽象的なアイコン、線、カード、矢印で構成。文字は読みやすく、装飾は控えめ。FRACTAL WORKOUTらしい黒いヘッダー帯と細いシルバーグレーラインを入れる。

## 保存先

image-gen2で生成した最終画像は以下へ保存済み。

```text

```
